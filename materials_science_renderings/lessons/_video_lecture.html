Video Lecture

The Video page layout includes an embedded YouTube video, which can scale in any browser or mobile device. Embedding videos in the page is a great way to present video content, while accompanying it with supporting context, explanations and activities.

Best Practices

Whenever possible, videos should be embedded in the content, rather than linked to. This simple stylization houses all the required information related to a video, including a title, the context or explanation of the video and its relevance to the learner, a link to the video's location and a downloadable transcript of the video.

Please note, it is best practice for accessibility to update the "title" attribute for the video "iframe". This is best accomplished by using the opening the source code editor, searching for "title of video" and entering the title of the video (and removing the square brackets).

Adding a Video

Inserting a video on your page is simple. Recognized video player formats are now automatically wrapped in the responsive wrapper demonstrated below. Once in the Edit HTML mode, you can use Insert Stuff > Enter Embed Code and paste your video's share embed code or Insert Stuff > Media Library to use a video currently uploaded to your instance's Media Library.

Currently supported video player formats:

  • YouTube
  • Vimeo
  • Brightspace Media Library
  • HTML5 <video>
