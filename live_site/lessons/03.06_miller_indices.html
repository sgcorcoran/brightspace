Miller Indices

Introduction to Miller Indices

Miller indices are a notation system used to describe the orientation of directions and planes in a crystal lattice. These indices are fundamental for understanding the geometry and properties of crystals.

Defining Directions and Planes

Crystallographic Directions

To define a direction in a crystal:

  1. Find the fractional coordinates of two points along the direction of interest. The tail and tip are often the most convenient.
  2. Subtract the coordinates of the tail point from the head point. This basically just shifts the vector to the origin.
  3. Clear fractions by multiplying the coordinates by the least common multiple of denominators and if needed reduce to the smallest integers.
  4. Enclose the numbers in square brackets, using an over bar for negative values and NO COMMAS. Example: \([1, 0, \overline{1}]\). We would read this as, "the direction one zero one bar"

Let's find the direction in the figure above following the steps we just defined.

  1. tail = (1,0,0) and tip = (1,1,1)
  2. (1,1,1) - (1,0,0) = (0,1,1)
  3. No fractions to clear so we still have (0,1,1)
  4. The direction is: [0 1 1]

Crystallographic Planes

To define a plane in a crystal:

  1. Identify the fractional intercepts of the plane with the crystal axes (a, b, c). If a plane does not intersect an axis, then either the plane is parallel to the axis or the plane just needs to be extended beyond the unit cell until it does intersect the axis (either positive or negative axis). If the plane is parallel to the axis, the intercept is taken as infinity.
  2. Take the reciprocal of each intercept.
  3. Clear fractions using the least common multiple of denominators but do not reduce the resulting integers.
  4. Enclose the result in parentheses and NO COMMAS, e.g., \((1 0 \overline{2})\).

Let's find the shaded plane in the previous figure.

  1. intercepts along \(a, b, c\) = \( 1, \infty, \infty\)
  2. taking reciprocals we get \(1, 0, 0\)
  3. No fractions to clear so we still have \(1, 0, 0\)
  4. The plane is: \((1 0 0)\)
