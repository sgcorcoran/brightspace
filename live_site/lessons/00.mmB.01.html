Granta EduPack: Advanced Materials Charts

Learning Objectives

By the end of this chapter, students should be able to:

  1. Create an advanced plot using indices (equations) of multiple properties.
  2. Apply material selection constraints using the Limit button.
  3. Filter materials based on crystal structure and phase at room temperature.

Accessing Advanced Charting in CES EduPack

1. Open Granta EduPack and navigate to the Elements database.

2. Click on the Chart/Select tab to open the Selection Project window.

3. Under "2. Selection Stages", select the Limit icon.

Limiting Materials Based on Crystal Structure, Phase and State

Using the Limit Button

1. Select the Limit icon to open the Selection Criteria panel.

2. Apply the following filters:

  • Crystal Structure: Select FCC, BCC, etc.
  • State at 300K: Select Metal or Non-metal.
  • Phase at 300K: Select Solid, Liquid, or Gas.

3. Click Apply.

Plotting an Equation Using the Advanced Button

1. Select the Chart/Index icon to open the Chart Stage window.

2. Select either the X-Axis or Y-Axis tab and then Click on the Advanced... button.

3. Enter a custom equation using the editor. Note: use the insert button to place a property into your equation.

For example, to plot Young’s Modulus/Density (E/ρ), the equation will look like: [Young's modulus at 300K] / [Density at 300K]

4. Click OK to apply the equation.

Refining the Plot

  • Adjust axes limits for better visualization if needed.
  • Adjust the window size to make sure the plot is more compact before copying.
  • Click on individual data points to label.
