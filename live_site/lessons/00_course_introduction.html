Welcome to MSE 2034 Elements of Materials Engineering

Course Structure

The course is organized into Modules, which are like chapters in a textbook. Each module contains:

  • Module Learning Objectives: Outline the key takeaways for the module.
  • Course Lessons: Cover the main content with text, videos, asides, and interactive elements.
  • Worked Example Lessons: Provide step-by-step calculations and applications of concepts.
  • Quizzes (Homework): Each module includes a quiz, which functions as your homework. You are allowed two attempts for most quizzes.

Key Features in Lessons

  • Interactive Elements: Tools like sliders, demos, or animations to help you visualize concepts.
  • Flip Cards: Concept checks using flip cards challenge your understanding. Click to reveal answers.
  • Videos: Short lessons provide deeper explanations. Examples include:
    • "Energy-Separation Plots" in Module 2
    • "Levels of Structure" in Module 3
  • Asides: Side sections provide additional details or clarifications on specific topics.
  • Downloadable Resources: Some lessons provide downloadable slides or supplementary materials.

Tips for Success

  • Read the Learning Objectives First: They set the stage for the content in the module.
  • Engage with Interactive Elements: These are designed to deepen your understanding.
  • Concept Checks:Spend time to answer the concept check questions before clicking on the flip card for the answer.
  • Attempt Quizzes Early: Use the two attempts to improve your learning and grade.
